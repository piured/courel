Judgment Courel.Judge.IJudge.EvalEndHoldEvent ( float  delta,
Hold  hold 
)

Judges the end of a Courel.Loader.Notes.Hold.

Parameters
deltaDelta time (in seconds) between the end holds's action time $v$ and the song time when the end hold event was produced.
holdHold to be judged.
Returns
If note is premature, return new instance of Courel.Judge.Judgment with Courel.Judge.Judgment.Premature set to true. If note is miss, return new instance of Courel.Judge.Judgment with Courel.Judge.Judgment.Miss set to true. If note is not premature nor miss, return new instance of Courel.Judge.Judgment with both Courel.Judge.Judgment.Miss and Courel.Judge.Judgment.Premature set to false. The sequencer will assume that the hold end was hit. You are not allowed to return null.
