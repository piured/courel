◆ GetBpms()

List< GimmickPair > Courel.Loader.IChart.GetBpms ( )

The BPMs definition for the chart.

Returns
Each Courel.Loader.GimmickSpecs.GimmickPair in the returning list is a definition of a BPM change, where Courel.Loader.GimmickSpecs.GimmickPair.Beat is the beat at which the BPM change occurs, and Courel.Loader.GimmickSpecs.GimmickPair.Value is the new BPM value. Charts without BPM changes (that is, only one BPM definition) must return a list with only one element, with Courel.Loader.GimmickSpecs.GimmickPair.Beat set to 0, and Courel.Loader.GimmickSpecs.GimmickPair.Value set to the BPM value. This method must not return null.

◆ GetSpeeds()

List< Speed > Courel.Loader.IChart.GetSpeeds ( )

The Speeds definition for the chart. The speeds gimmick changes the effective scrolling speed at a given beat by a rate defined by the speed value. The speed changes linearly according to the span of time defined by the speed gimmick. For further details, see the wiki.

Returns
Each Courel.Loader.GimmickSpecs.Speed in the returning list is a definition of a speed gimmick, where Courel.Loader.GimmickSpecs.Speed.Beat is the beat at which the speed gimmick occurs, Courel.Loader.GimmickSpecs.Speed.Value is the new speed value, Courel.Loader.GimmickSpecs.Speed.SpanTime is the span of time for the speed transition from the previous speed to the new speed, and Courel.Loader.GimmickSpecs.Speed.SpanTimeType is the type of span time. Charts without speed gimmicks must return a list with only one element, with Courel.Loader.GimmickSpecs.Speed.Beat set to 0, Courel.Loader.GimmickSpecs.Speed.Value set to 1, Courel.Loader.GimmickSpecs.Speed.SpanTime set to 0, and Courel.Loader.GimmickSpecs.Speed.SpanTimeType set to Courel.Loader.GimmickSpecs.SpanTimeType.Beats. This method must not return null.
