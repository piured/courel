Input events that notes can react to.

Enumerator
Tap 

A note reacting to a Tap event will be judged after Courel.Sequencer.Tap is called.

Hold 

A note reacting to a Hold event will be judged according to the Courel.Input.IHoldInput state of the note's lane.

A note reacting to a Lift event will be judged after Courel.Sequencer.Lift is called.
